# back_up

このフォルダには，開発途中のコードや試作した機能を保存しています．

本プロジェクトは，最初から完成形を作るのではなく，小さな機能ごとに動作確認しながら段階的に開発しました．開発の流れは以下の通りになります．

1. Webカメラ映像の取得と表示
2. 左右反転表示
3. テキスト表示
4. MediaPipe用画像への変換
5. 手の検出
6. ランドマークの点の描画
7. ランドマークの線の描画
8. 距離計算による指本数判定
9. 左右の手の分離
10. リファクタリング(各コードの関数化)
11. リファクタリング(main関数の実装)
12. データ収集
13. MLPによる指本数認識
14. 四則演算への拡張
15. クイズモードの試作

`14_def_calc_quiz.py` は，ジェスチャーで計算問題に答えるクイズモードを作るための試作コードです．
クイズモードは本体には実装していません．リアルタイム映像では同じジェスチャーが複数フレームにわたって検出されるため，今後は状態管理や入力ロックを設計したうえで実装する予定です．
